CSS Typeface Process

Project Brief

Create a typeface (min 26 letters) using only HTML and CSS.

Approaching the Project

I first tried to figure out how I can approach this project. Knowing that I can create grids in CSS, I can base my typeface under a grid system. The first thing that came to mind was Armin Hoffmann's circle grid from his design manual. I can create letterforms using the circle guide. I can also create something more interesting by shrinking some of the circles.

Another way I can create my typeface is use a specific set of repeated parts/forms. Since I can create a CSS class for each specific part. I can use maybe 3 or 5 parts, or just one part (a motif).

Below are some explorations. Eventually, I focused more on the second approach (using repeated forms), because creating the irregular forms using the circle grid seemed extremely challenging to execute with CSS. And I think using the repeated forms approach brings out the true potential of CSS more than the circle grid approach as it is more efficient.

Draft

Critique:

  • Letter S - not so legible
  • Delicate serifs compared to the other strokes
  • Letter Q - make descender delicate
  • Some of the shapes (especially curves) are not flush

Refinement

I also created another version with thinner serifs, but I think the serifs still looks too delicate.

I added numbers. I think the numbers without serifs are better as they are more differentiable from the letters.

Critique:

  • The circles on the serifs don't really connect with the stems
  • This is the same with the square serifs before
  • Maybe instead of rigid corners, try curves
  • Some of the letters aren't so consistent with others
  • R and O with the angle created by the two curves looks "bulky"
  • M and N looks like they are another set compared to R and O
  • Can try using diagonal lines instead of just restricting to horizontal and vertical arrangements
  • Maybe that can help make the V more pointed at the bottom
  • Serifs for the numbers
  • Play around with the serifs by removing the circles on the serifs

Final

Due to time constraints, I decided to refine only the letters and the hyphen, as I only needed these for the Web Typographic Poster.

I got rid of most of the curves as these are the parts that made the letterforms look bulky. I replaced them with the small curved form that is used for the serifs. I also changed up the letters like E, F, C, L, J, which originally had horizontal thick stems, to vertical. I think that is more consistent with the letters like M, N, H, A, where they have the vertical thick stems.

I ended up going with the one with the thin serifs, as I think the high contrast makes the letters more elegant.
